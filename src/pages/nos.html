The Project

nOS is a virtual operating system that allows users to connect to the Blockchain Powered Open Internet and Smart Economy.

In May 2018 nOS decided to launch a community UI design competition for its nOS client platform. The competition required participants to develop UI concepts for the nOS platform that integrated an nOS menu with a web browser, login page, DApp discovery dashboard, settings and token wallets.

My approch was to come up with a clean user interface which would not dostract the user from the main tasks and appling common UI patterns.

nOS wallet

Sending and receiving cryptocurrencies can be a daunting task, especially for new users. For blockchain technology to be more adopted the experience has to be as familiar to users and as seamless as possible. Each screen is designed to guide the user and provide feedback for every step of the process.

1.0a / Enter recipient address

User has the option of either choosing existing address or entering a new one.

1.0b / Enter recipient address

User choose to enter a new public address.

2.0 / Token and amount

User selects the token and amount to send.

3.0 / Confirm trasaction details

User confirms details are correct before sending.

4.0 / Transaction completed

Confirmation that tokens were successfully sent.
